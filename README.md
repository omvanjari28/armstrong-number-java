 Java Practice Programs

This project contains small Java practice programs for learning basic programming concepts such as input handling, conditionals, loops, patterns, factorials, calculators, pyramids, and number checks.

## Main Program: Armstrong Number

The armstrong.java program checks whether a user-entered number is an Armstrong number.

An Armstrong number is a number that is equal to the sum of the cubes of its digits.

Example:

text
153 = 1^3 + 5^3 + 3^3
153 = 1 + 125 + 27
153 = 153

Example Output:
enter number:
153
153 is an armstrong number

enter number:
165
165 is not an armstrong number
